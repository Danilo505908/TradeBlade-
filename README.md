# TradeBlade - Платформа копи-трейдинга

Веб-сайт платформи TradeBlade, створений на основі макету Figma з використанням HTML, CSS та JavaScript.

## Структура проєкту

```
TradeBlade-/
├── index.html              # Головна HTML сторінка
├── styles/
│   └── main.css           # Основні стилі
├── js/
│   └── main.js            # JavaScript функціонал (оптимізований та організований)
├── assets/
│   └── images/            # Зображення (тільки використані файли)
│       ├── logo-header.svg
│       ├── logo-footer.svg
│       ├── hero-image.png
│       ├── feature-*.svg
│       ├── tradeblade-feature-*.svg
│       └── інші використані зображення
└── README.md
```

## Особливості

✅ **Повна відповідність макету Figma** - структура та дизайн максимально наближені до оригінального макету  
✅ **Адаптивний дизайн** - сайт коректно відображається на всіх пристроях  
✅ **Інтерактивність** - плавна прокрутка, анімації, мобільне меню  
✅ **Оптимізація** - швидке завантаження та плавна робота  

## Секції сайту

1. **Header** - навігація з логотипом, меню та кнопками реєстрації/входу
2. **Hero** - головна секція з заголовком, описом та формою реєстрації
3. **Цифры** - секція зі статистикою та графіком
4. **О компании** - інформація про компанію та форма спробувати безкоштовно
5. **tradeblade это** - картки з можливостями платформи
6. **Как начать** - покрокова інструкція початку роботи
7. **Footer** - підвал з посиланнями

## Запуск

Просто відкрийте `index.html` у браузері або використайте локальний сервер:

```bash
# Python
python -m http.server 8000

# Node.js
npx http-server -p 8000
```

Потім відкрийте `http://localhost:8000` у браузері.

## Використані технології

- **HTML5** - семантична розмітка
- **CSS3** - сучасні стилі з CSS змінними
- **Vanilla JavaScript** - без додаткових залежностей
- **Figma MCP** - завантаження зображень з макету
